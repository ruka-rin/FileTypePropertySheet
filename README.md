# FileTypePropertySheet

Add filetype detector property sheet in system property shell

Use [ServerManager](https://github.com/dwmkerr/sharpshell/releases/) to install server assembly `FileTypePropertySheet`.

使用 [ServerManager中文版](https://github.com/ruka-rin/ShellContextMenu/blob/main/ServerManager/ServerManager%E4%B8%AD%E6%96%87%E7%89%88.zip) 安装服务程序集`FileTypePropertySheet`。
